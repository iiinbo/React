1. 값타입 변수 - 값을 저장한다
2. 참조타입 변수 - 값이 있는 곳 주소를 저장한다. 이를 참조라 함
3. 프로그램을 가동하면 할당되는 메모리는 3부분으로 나뉜다.
4. static / stack / heap 공간(논리적 단위임. 실제 물리적인 공간 아님)

5. static : 메서드 또는 함수, 상수 등이 저장 [관리자 : system]

6. stack : 함수 호출 시 호출된 함수가 차지하는 메모리.
           들어간 순서와 반대로 나오는 구조다.
            일반 변수들이 이 공간에 확보 [관리자 : system]
            int a; //값 변수. 스택 공간에 자리잡고 데이터도 스택 공간이다.
            a = 9; //9라는 상수 값이 a메모리에 저장된다.

7. heap : 객체나 배열이 저장된다.
          이 공간에 메모리를 확보하고 데이터를 저장하면 이 공간에 대한 주소를 반환한다.

person p1 : 변수 자체는 스택에 저장된다. 이 변수엔 객체의 주소가 저장되므로 참조변수라 부른다.
            아직 데이터를 저장할 공간은 존재하지 않는다.
= new Person(); : 힙공간에 person 객체를 생성하고, 생성한 객체의 첫번째 번짓값을 p1에 할당한다.
                  만일 메모리가 부족해서 객체 생성을 못할 경우 null로 채워진다.
